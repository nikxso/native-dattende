# Native-Dattende 📍📸

Native-Dattende is an open-source **React Native attendance application** built with **Expo** and **Firebase**. The app enables students, teachers, and organizations to manage attendance efficiently using GPS location tracking, image verification, and role-based access control.

Users can check in and check out seamlessly while the system records attendance time, location, and images in real time.

### 📥 [Download Latest Release](https://github.com/nikxso/native-dattende/releases)

---

## Features

### 🏠 Home Page

* **User-Friendly Interface**: A clean and intuitive interface for quick attendance marking.
* **Real-Time Data**: Users can view their attendance status, check-in time, and check-out time.
* **Location Tracking**: GPS location is captured during attendance for verification.
* **Photo Verification**: Attendance images are securely stored for record keeping.

## Additional Information

### 🛠 Technology Stack

* **React Native (Expo)** – Cross-platform mobile application development.
* **Firebase Firestore** – Real-time database and attendance records.
* **Firebase Storage** – Secure image storage.
* **React Navigation** – Navigation and screen management.
* **AsyncStorage** – Local session persistence.
* **Expo Location** – GPS-based attendance verification.
* **NetInfo** – Internet connectivity monitoring.
